本拡張機能について

本拡張機能「OOKABE GLASS メールワイズ拡張機能」は、OOKABE GLASS株式会社が社内メンバー向けに限定公開している Chrome ブラウザ拡張機能です。サイボウズ メールワイズの画面上に、社内で運用している業務ツールへの起動ランチャーを追加します。

収集する情報

本拡張機能は、個人を特定できる情報を一切収集しません。以下の情報も収集しません。

  • 個人識別情報 (氏名、メールアドレス、住所、電話番号等)
  • 認証情報 (パスワード、認証トークン等)
  • 位置情報
  • Web の閲覧履歴
  • ユーザーの操作内容 (アナリティクス、トラッキング等)

ローカルに保存する情報

以下の情報を、chrome.storage.local を用いてユーザーのブラウザ内にのみ保存します。外部サーバーへ送信することはありません。

  • 本拡張機能および各機能の有効/無効の設定状態
  • UI 表示に関する設定値

これらの情報は、本拡張機能をアンインストールすると削除されます。

外部通信について

本拡張機能自体は、外部サーバーへユーザーデータを送信しません。本拡張機能が起動ランチャーからアクセスする社内業務ツールは、それぞれ独自のプライバシーポリシーおよびアクセス制御下にあり、社内ネットワークからのみアクセス可能です。

第三者への提供

収集した情報を第三者へ提供することはありません (そもそも収集していません)。

権限の使用目的

本拡張機能が要求する権限は、いずれも本拡張機能の中心機能を提供するためにのみ使用します。

  • storage: 設定値の保存
  • host_permissions (https://gmcc.cybozu.com/*): メールワイズ画面上での UI 要素の表示

ポリシーの変更

本ポリシーは、必要に応じて予告なく変更される場合があります。変更があった場合は、本ページ上部の最終更新日を改訂します。
