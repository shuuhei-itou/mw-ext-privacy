OOKABE GLASS メールワイズ拡張機能
プライバシーポリシー

最終更新日: 2026年7月13日

本ポリシーの対象

本ポリシーは、OOKABE GLASS株式会社が社内メンバー向けに限定公開している、サイボウズ メールワイズ関連の Chrome ブラウザ拡張機能に適用されます。対象拡張機能は、メールワイズ画面上への業務ツール起動ランチャーの追加、および処理待ちメールの新着・警告のデスクトップ通知を行います。

収集する情報

本拡張機能は、個人を特定できる情報を一切収集しません。以下の情報も収集しません。

  • 個人識別情報 (氏名、メールアドレス、住所、電話番号等)
  • 認証情報 (パスワード、認証トークン等)
  • 位置情報
  • Web の閲覧履歴
  • ユーザーの操作内容 (アナリティクス、トラッキング等)

ローカルに保存する情報

以下の情報を、chrome.storage.local を用いてユーザーのブラウザ内にのみ保存します。外部サーバーへ送信することはありません。

  • 本拡張機能および各機能の有効/無効の設定状態
  • UI 表示に関する設定値
  • 通知対象とする担当者/確認者のフィルタ設定、監視URL、チェック間隔
  • 新着判定に用いる既読状態

これらの情報は、本拡張機能をアンインストールすると削除されます。

外部通信・メール情報の取り扱いについて

本拡張機能は、ユーザーがログイン済みのサイボウズ メールワイズ (https://gmcc.cybozu.com) から、処理待ちメールの一覧情報を取得 (fetch) する場合があります。取得した情報は、ユーザーのブラウザ内での一覧表示および新着・警告のデスクトップ通知にのみ使用し、外部サーバーへの送信、第三者への提供、販売は一切行いません。

上記メールワイズを除き、本拡張機能は外部サーバーへユーザーデータを送信しません。本拡張機能が起動ランチャーからアクセスする社内業務ツールは、それぞれ独自のプライバシーポリシーおよびアクセス制御下にあり、社内ネットワークからのみアクセス可能です。

第三者への提供

収集した情報を第三者へ提供することはありません (そもそも収集していません)。

権限の使用目的

本拡張機能が要求する権限は、いずれも本拡張機能の中心機能を提供するためにのみ使用します。

  • storage: 設定値・フィルタ・既読状態の保存
  • alarms: 処理待ちメールを一定間隔で確認する定期チェックのスケジューリング
  • notifications: 新着メール・警告日時の到来を知らせるデスクトップ通知
  • host_permissions (https://gmcc.cybozu.com/*): メールワイズ画面上での UI 表示、および処理待ちメール一覧の取得

ポリシーの変更

本ポリシーは、必要に応じて予告なく変更される場合があります。変更があった場合は、本ページ上部の最終更新日を改訂します。
